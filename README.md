
Приложение погоды для ios - swift + uikit

## Что внутри?

**Главный экран** — Текущая погода, влажность, ветер,и прогноз на 5 дней.

![Main Weather Screen](./Simulator%20Screenshot%20-%20iPhone%2016%20Pro%20-%202026-03-25%20at%2016.51.12.png)

**Сохранённые города** — Отображенные локации, Свайп влево — удалить.

![Saved Locations](./Simulator%20Screenshot%20-%20iPhone%2016%20Pro%20-%202026-03-25%20at%2016.51.08.png)

**Поиск** — поиск города по названию, подсказки из Open-Meteo API.

![Search](./Simulator%20Screenshot%20-%20iPhone%2016%20Pro%20-%202026-03-25%20at%2016.50.13.png)

## Технологии

- **UIKit** + **SnapKit** — вёрстка
- **MVVM** + **Coordinator** — архитектура
- **Open-Meteo API** — апи
- **async/await** — асинхронность
- **UserDefaults** — сохранение городов
